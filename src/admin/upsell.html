UPSELL&UPGRADE

UPSELL

Повышение категории номера за дополнительную плату.

  1. Если upsell с номера повышенной комфортности: Room Type UKN/UTN, RTC - SKN/STN, Packages → new → package → UPSELL1 GST to SUP, в комментариях пишем UPSELL BY FO**.
  2. Если upsell с номера категории полулюкс: Room Type JKN, RTC - SKN, Packages → new → package → UPSELL2 GST to JST, в комментариях пишем UPSELL BY FO**.
  3. Если upsell с номера категории люкс: Room Type 1KN, RTC - SKN, Packages → new → package → UPSELL3 GST to 1ST, в комментариях пишем UPSELL BY FO**.

Разница в стоимости:

  • SKN>UKN - 1000р.;
  • SKN>JKN - 3000р.;
  • SKN>1KN - 5000р..
  • UKN>JKN - 2000р.;
  • UKN>1KN - 4000р.
  • JKN>1KN - 2000р.

За каждый сделанный Upsell + к премии:

  • за UKN/UTN - +100р./ночь;
  • за JKN - +200р./ночь;
  • за 1KN - +300р./ночь.

Upgrade

Комплиментарное повышение категории номера: делается все тоже самое, только не прикрепляется Package, просто фактический Room Type и RTC который был, стоимость оставляем изначальную.

Если гость заехал и хочет повысить категорию: RTC оставляем, Room Type меняем на желаемую категорию. Заселяем, принимаем оплату, начисляем маркет кодом повышение категории номера.
