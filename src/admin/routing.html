Routing

Рутинг с одного номера на другой номер:

В случае, если один номер решил оплатить за несколько номеров:

На одном номере принимаем общую оплату за нужное количество номеров, на других номерах, на которых нет оплат, заходим в Options → Routing → Route to Room пишем номер комнаты, на котором принята общая оплата → Transactions: BB.

В комментариях номера, на котором принята оплата пишем, за какой номер оплатил. Т.к. могут возникнуть неприятные ситуации, связанные с сокращением брони.

Соответственно, на номерах, на которых нет оплат, должен стоять рутинг. На номере, на котором принята оплата, рутинга быть не должно.

Comfort Booking

Если оплаченное бронирование сделано через Comfort Booking (в подтверждении бронирования, в поле “Способ оплаты” будет указан “Банк. карта: Эквайринг ComfortBooking (Российская банк. карта)”, а в поле “Внесена предоплата” будет указана вся сумма проживания):

Меняем язык на RU, Ставим в Agent: Comfort Booking, рутинг не ставим. В Res. Type ставим DR, Payment ставим CA, в комментариях прописываем: по выезду закрыть на site payment 9081.

Yandex Travel/Яндекс.Путешествия

Если оплаченная бронь падает с Yandex Travel (в подтверждении бронирования, в поле “Внесена предоплата” будет указана вся сумма проживания), то рутинг не ставим. В Agent автоматически будет Yandex Verticali. В Res. Type ставим DR, Payment ставим CA, язык RU, в комментариях прописываем: по выезду закрыть на yandex payment 9019.

Если бронь от Яндекс с предоплатой в 1р. то с таким бронированием можно ничего не делать.

Best Eastern Hotel

Если бронирование оплачено с ООО “БЭСТ ИСТЕРН ОТЕЛЬ”, в брони агент будет указан Academ Service, меняем его на Best Eastern Hotel.

Меняем язык на RU, далее Options → Routing → Window → стрелку возле Name → выбираем агента Best Eastern Hotel → Transactions: BB → Ok

В брони Res. Type: TA; Payment: DB


Если, Вы поставили в поле Agent: Best Eastern Hotel, поставили на него рутинг и приходит коррекция на бронирование, то Agent автоматически снова изменится на Academ Online. Необходимо будет поменять снова на Best Eastern Hotel.

Ostrovok

  1. Если в подтверждении бронирования в способе оплаты написано Безналичный расчет (Оплачено на сайте канала) и в самом низу подтверждения будет прописана компания ООО Бронирование гостиниц, то в поле Agent остается Ostrovok. Поставить рутинг на Ostrovok. Res. Type должен быть TA, Payment должен быть DB.
  2. Если в подтверждении бронирования в способе оплаты написано Безналичный расчет (Оплачено на сайте канала) и в самом низу подтверждения будет прописана компания ООО Агентика Трэвл, то в поле Agent удаляем Ostrovok и пишем туда Agentika Travel. Поставить рутинг на Agentika Travel. Res. Type должен быть TA, Payment должен быть DB.
  3. Если в подтверждении бронирования в способе оплаты написано Банковская карта и ООО Бронирование гостиниц, то рутинг ставить не нужно. В поле Agent остается Ostrovok. Res. Type должен быть TA, Payment должен быть CA.

X5 retail group и Baltika

Ставим в Agent Aeroclub, ставим рутинг на Aeroclub.

Другие каналы

Если бронирование с других каналов (Bronevik, HB-Pro и др.):

Меняем язык на RU, в Agent уже будет выставлен автоматически, Option → Routing → Window → стрелку возле Name → выбираем агента, смотрим чтобы был указан AR № → Transactions: BB → Ok

Routing на PM

Выбирай тот, у которого есть A/R Number
