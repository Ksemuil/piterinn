Штраф NO SHOW по DB или поздняя аннуляция

Если гость заехал после ночного аудита, то начисляем предыдущую ночь через Post Rate Code. Перед этим, сверить стоимость предыдущей ночи. Подробнее читай тут.


Если гость не заехал до 12:00 следующего дня заезда:

  • Update Reservation → ищем бронь → запоминаем сумму штрафа в Rate Info → Options → Add On → Room Type: PM → Profile → New → Last Name оставляем фамилию гостя → First / Middle пишем No show fee → Ok.
  • В самой брони ставим: 0 Nights, 0 Adults, Rate Code - Z-PM, Market: OTH (Other Room Revenue) → Source: HSE (HOUSE USE) → в Room приписываем любой свободный PM.
  • Далее ищем созданный PM в Arrivals, заселяем, заходим в биллинг, начисляем сумму штрафа маркет кодом 3018 → Supplement: Штраф за незаезд ДД.ММ, Reference: Штраф за незаезд ДД.ММ → Check out


Открываем папку No show, смотрим последний распечатанный отчет, ищем оштрафованную только что бронь и подписываем, что номер оштрафован.

Штраф за NO SHOW или позднюю аннуляцию гарантированных бронирований (СС) С САЙТА

Бронирования с сайта с методом оплаты «гарантия банковской карты» в Оперу падают без номера банковской карты.
Такие брони, как обычно, отправляем в No Show в случае незаезда.
Оштрафовать можно будет в TL: отчеты → отчеты по броням.
Находим нужное бронирование. Нажимаем “Получить оплату”.

Если оплата прошла, то создаем РМ и начисляем штраф, закрываем на site payment.

Штраф за NO SHOW или позднюю аннуляцию гарантированных бронирований (СС) НЕ С САЙТА

Если гость заехал после ночного аудита, принимаем оплату за проживание с учетом предыдущих суток и начисляем незаезд через Post Rate Code.


Если гость не заехал до 12:00 следующего дня заезда:

В данном случае необходимо списать средства с банковской карты гостя, которую он указал для гарантии при бронировании. Данные карты смотрим в брони.

Мануальный терминал: ручной ввод → вводим номер карты, срок действия, сумму → владелец карты выбираем не присутствует.

Если средств недостаточно, чек крепим к отчету No Show.


Если средства списались, то создаем PM для начисления штрафа:

Update Reservation → ищем бронь → Options → Add On → Room Type: PM → Profile → New → Last Name оставляем фамилию гостя → First / Middle пишем No show fee → Ok.

В самой брони ставим: 0 Nights, 0 Adults, Rate Code - Z-PM, Market: OTH (Other Room Revenue) → Source: HSE (HOUSE USE) → в Room приписываем любой свободный PM.

Далее ищем созданный PM в Arrivals, заселяем, заходим в биллинг, начисляем оплату с мануального чека через Payment (Visa M, Мир М и т.п), начисляем сумму штрафа маркет кодом 3018 → Supplement: Штраф за незаезд ДД.ММ, Reference: Штраф за незаезд ДД.ММ → Check out

Фолио с чеком крепим к отчету No Show, в самом отчете отмечаем, что номер оштрафован.

Штраф Best Eastern Hotel

Если гость приехал после ночного аудита, т.е. бронь перешла в No Show, то в Billing, через Post Rate Code, начисляется предыдущая ночь.


Если гость не заехал до 12:00 следующего за днем заезда, передать отделу бронирования, т.к. бронирования от Best Eeastern Hotel штрафуются не на ту сумму, которую можно увидеть в заявке или через Rate Info.

Штрафы групп

Если от группы не заехал номер, можно начислить штраф маркет кодом прямо на PM. Узнать номер PM можно, к примеру, нажав на Routing на одном из номеров группы.
