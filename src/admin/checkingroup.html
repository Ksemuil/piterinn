Check In групп

ЗАСЕЛЕНИЕ ГРУПП

Ждем, когда перед стойкой соберется вся группа. Сообщаем информацию по гостинице.

Если у группы оставлены вещи в комнате хранения багажа, желательно сразу открыть для них дверь (в случае большой загрузки, можно попросить об этом сотрудников СЭБиК).


Проверяем, все ли номера готовы. Для этого заходим в Room Assignments, смотрим на статус номера (Clear/Dirty).

Если у кого-то из группы остался грязный номер, звоним в поэтажную службу, уточняем. Если номер готов, поменять статус. Если номер не готов, ищем другой.


Если группа большая, собираем паспорта, в паспорта вкладываем Passport Tag, куда записываем номер комнаты. Сообщаем гостям, что паспорта необходимо оставить для регистрации примерно на пол часа.

Желательно отсортировать оставленные паспорта по номеру комнат, так быстрее получится найти нужный паспорт и не заставлять гостей ждать.

Сверяем фамилии гостей в паспортах с фамилиями в Опере, находим необходимый ключ, проверяем, что на Key-Wallet указаны нужные фамилии, отдаем ключи гостям.

Детские группы

Если заселяется детская группа:

  • записываем ФИО руководителя;
  • номер комнаты руководителя;
  • контактный номер телефона руководителя;
  • уточняем время выезда группы;
  • предупреждаем старших, что электро приборы (чайные наборы и гладильные наборы) могут быть выданы только в номера, где есть взрослые;
  • зовем сотрудника СЭБиК, чтобы провели инструктаж для детей;
  • коллективам разрешено репетировать в тренажерном зале, но вся запись ведется организованно через старшего смены охраны.

В это время собираем все документы, сверяем количество документов с количеством людей в Group Rooming List. Отдаем все ключи руководителю. Проговариваем ему всю информацию о гостинице.

Далее, чтобы заселить всю группу:

  1. Front Desk → Room Assignment → Advanced → Assigned Rooms → Group: Пишем название группы → Search → отсортировать по номерам.
  2. На Пробел можно выделить необходимое количество гостей и заселить их через кнопку Check In.
  3. В появившемся окне выбираем Do not Cut Keys to any guest и Do not print registration cards to any guest, после чего нажимаем Check In.
Если выделить одного человека из номера и нажать Check In, то заселятся все равно все гости из этого номера. Если заехал один человек из номера, а другой будет позже, то заселяем его через Arrivals.

После заселения группы, делаем сканы документов. Заполняем в Контур. Отдаем паспорта гостям по их требованию.


Не оставляем копии документов таким образом, чтобы их было видно другим гостям.
