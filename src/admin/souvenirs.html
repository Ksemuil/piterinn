Сувениры

Ключи от шкафчиков находятся в кассе FO1, номер каждого ключа соответствует шкафчику если считать слева направо.

На каждом товаре есть наклейка с кодом товара. Также код товара можно увидеть на ценнике.

Кнопки на сувенирной кассе:

  • ПИ - Промежуточный итог;
  • ИТ - Итог;
  • ВЗ - Возврат;
  • + (плюс) - Переключает по меню.

Открытие сувенирной кассы

С - на экране появится “Кассовый режим”, если нет, то пролистать до него на “РЕЖ”, нажать “ИТ”, на экране появится “Открыть смену”, нажать “3”. Чек открытия можно выкинуть.

Оплаты

Если оплата наличными:

  • Нажимаем КОД → вводим код с ценника или наклейки на товаре → ПИ → еще раз ПИ → ИТ. Выйдет кассовый чек, его можно отдать гостю.

Если несколько товаров наличными:

  • КОД → вводим код с ценника или наклейки на товаре → ПИ → снова КОД → вводим код второго товара → ПИ → ПИ → ИТ. Выйдет кассовый чек, его можно отдать гостю. (при нажатии второй раз ПИ отображается общая сумма за товары)

Если оплата по карте:

  • Нажимаем КОД → вводим код с ценника или наклейки на товаре → ПИ → еще раз ПИ → 00 → ИТ. Выйдет кассовый чек. На терминале вводим нужную сумму, гость прикладывает или вставляет карту, выходят 2 терминальных чека, 1 из них крепим с чеком из кассы и отдаем гостю, 2-й терминальный чек кладем в кассу.

Если несколько товаров по карте:

  • КОД → вводим код с ценника или наклейки на товаре → ПИ → снова КОД → вводим код второго товара → ПИ → ПИ → 00 → ИТ. Выйдет кассовый чек. На терминале вводим нужную сумму, гость прикладывает или вставляет карту, выходят 2 терминальных чека, 1 из них крепим с чеком из кассы и отдаем гостю, 2-й терминальный чек кладем в кассу. (при нажатии второй раз ПИ отображается общая сумма за товары)

Прежде чем пробивать чек безнал. по кассе, дождитесь пока оплата пройдет на терминале. Иногда бывает так, что средств на карте недостаточно и гость решает расплатиться наличными. Лишние возвраты никому не нужны.

Возврат

Возврат наличные:

  • Нажимаем ВЗ → + → ИТ→ КОД → вводим код товара с ценника или наклейки → ПИ → ПИ → ИТ

Возврат по карте:

  • Нажимаем ВЗ → + → ИТ→ КОД → вводим код товара с ценника или наклейки → ПИ → ПИ → 00 → ИТ → ИТ

При возврате по карте, может потребоваться произвести возврат не только по кассе, но и по терминалу. Для этого потребуется карта гостя, с которой была произведена оплата. Пароль для возврата по терминалу 0000.

Закрытие смены

Для закрытия кассы необходимо снять три отчета:

Сначала снимаем Нефиск. отчет ККН (аналог Х отчета), смотрим сколько прошло по кассе Наличными и Безналичными, считаем чеки, считаем наличку, считаем размен (размен должен составлять 7000р.), если все сходится можно закрыть смену и снять сверку итогов на терминале.

Заполняем журнал.

Подписываем конверт, пишем дату и надпись СУВЕНИРЫ, конверт скидываем в сейф в кабинете охраны.


Ночная смена пополняет сувениры. Но можно помочь коллегам и сделать это заблаговременно в любую смену.
