Профиль гостя

У гостей с Карельской пропиской, профиль гостя заполняем обязательно

Если гость уже заселялся ранее, нажимаем Options → Profile → Select Another Profile, вводим фамилию и имя гостя, ищем нужный профиль.

Если гость у нас не проживал или вы не можете найти его профиль, то заполняем данные самостоятельно. Нажав на копку рядом с Last Name, с изображением планеты, откроется окно Alternate Name, заполняем на кириллице графы Alt. Name, Alt. First, Alt. Language выставляем RU, заполняем отчество гостя в Alt. Env. Greeting.


Именно после заполнения этих полей, в квитанциях и счетах будет указана фамилия и имя гостя на кириллице.


Далее нажимаем на кнопку More Fields, которая находится над полем Last Name. В открывшемся окне, заполняем поля согласно документу гостя: ID Type выбираем PAS для паспорта или UNKNOWN для свидетельства о рождении, ID Number, ID Issued Date, ID Issued by RU, Gender RU, Birth Place, City / Nas. punkt RU/ Address RU



Также, заполнив полностью профиль гостя, при необходимости, можно распечатать Форму №5: Options → Register Card → Forma-5. Выглядит она следующим образом:

Shares

Если бронь сделана на 2/0.

Заходим в бронь, Options → Shares → Combine → далее на вопрос “Do you want to copy payment details” отвечаем No. Заполняем: Name, First Name, Adults ставим 0, Res. Type: NG, Payment: CA. Обязательно ставим галочку No Post.



Если бронь сделана на 1/0 и делается подселение.

Принимается оплата 1000р. за ночь. Далее все то же самое, что и при брони 2/0.

Далее, меняем бронь на 2/0, сохраняем старый тариф. И только после этого делаем ключи.


У шера автоматически создается EN язык, меняй в брони язык на RU.

Если на одну фамилию забронировано 2+ номеров, то при смене профиля на одном, поменяются профили на других. Чтобы этого не было, в каждой брони, в профиле нажимаем New.

Раздельные счета

Смотрим, сколько всего должны оплатить. Принимаем 2 оплаты по половине, чтобы было два чека. Делаем каждый шер по 1/0. Далее либо заходим в одну бронь Options → Shares → Split. Каждая ночь у каждого гостя должна располовиниться.

Либо, в случае, если стоимость за ночь меняется, то так сделать не получится. В таком случае заходим в каждый шер, Daily Details и меняем каждую ночь пополам вручную, ставим Fixed Rate.
