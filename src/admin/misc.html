Miscellaneous

Reports

Служит для поиска и печати различных отчетов.


Заходим в Miscellaneous → Reports.

В появившемся окне пишем название отчета. Достаточно первых букв.

  1. Отчеты для охраны:
    • Брони: Arrivals: Detailed → Выключить Traces, Notes, Preferences, sort order by: Alphabetical;
    • Список проживающих: Guest In House → Room Order → Print.
  2. Отчет для ресторана: Guest In House - Room Order → Print
  3. Отчет для выезда (если потребуется): Departures: Checked Out.
