Автозапуск

Инструкция актуальна для FO1 и FO2, где установлена ОС Windows 7.

  1. Открываем меню "Пуск", где находим папку "Startup"
  2. Жмём по папке правой кнопкой мыши, выбираем пункт "Open"
  3. Копируем сюда ярлыки тех программ, которые должны запускаться автоматически при входе в учётную запись: Document Reader, Google Chrome, Internet Explorer, ppServer.
  4. В браузере Chrome выбираем: Настройки → Запуск Chrome → Заданные страницы → Добавить страницу. Добавляем ссылки:
    1) https://e.mail.ru/inbox/
    2) https://piterinn.com/booking/
    3) https://fms.kontur.ru/Pending
